Detailed Description

A utility class for SDBM to represent an integer with potentially infinite positive value.

This uses the largest value of int64_t to represent infinity and redefines the arithmetic operators so that the infinity "saturates": inf + x = inf, inf - x = inf. If a sum of two finite values reaches the largest value of int64_t, the behavior of IntInfty is undefined (in practice, it asserts), similarly to regular signed integer overflow.
