Detailed Description

template<typename Op>
struct mlir::edsc::intrinsics::ValueBuilder< Op >

Helper variadic abstraction to allow extending to any MLIR op without boilerplate or Tablegen.

Arguably a builder is not a ValueHandle but in practice it is only used as an alias to a notional ValueHandle<Op>. Implementing it as a subclass allows it to compose all the way to Value. Without subclassing, implicit conversion to Value would fail when composing in patterns such as: select(a, b, select(c, d, e)).
