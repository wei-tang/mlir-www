171 /// An IntegerValueSet is an integer set plus its operands.
172 // Both, the integer set being pointed to and the operands can change during

175  /// Constructs an integer value set from an affine value map.

179  /// Returns true if this integer set is determined to be empty. Emptiness is
180  /// checked by by eliminating identifiers successively (through either
181  /// Gaussian or Fourier-Motzkin) while using the GCD test and a trivial
182  /// invalid constraint check. Returns 'true' if the constraint system is found
183  /// to be empty; false otherwise. This method is exact for rational spaces but
184  /// not integer spaces - thus, if it returns true, the set is provably integer
185  /// empty as well, but if it returns false, it doesn't necessarily mean an
186  /// integer point exists in it. This method also returns false where an
187  /// explosion of constraints is detected - due to the super-exponential
188  /// worse-case complexity of Fourier-Motzkin elimination (rare for realistic
189  /// problem cases but possible for artificial adversarial or improperly
190  // constructed ones), this method returns false conservatively.
191  bool isEmpty() const;

193  bool getNumDims() const { return set.getNumDims(); }
194  bool getNumSymbols() const { return set.getNumSymbols(); }
195 
196 private:
197  // The set pointed to may itself change unlike in IR structures like
198  // 'AffineCondition'.
199  MutableIntegerSet set;
200  /// The SSA operands binding to the dim's and symbols of 'set'.
201  SmallVector<Value, 4> operands;

204 /// A flat list of affine equalities and inequalities in the form.
205 /// Inequality: c_0*x_0 + c_1*x_1 + .... + c_{n-1}*x_{n-1} >= 0
206 /// Equality: c_0*x_0 + c_1*x_1 + .... + c_{n-1}*x_{n-1} == 0
207 ///
208 /// FlatAffineConstraints stores coefficients in a contiguous buffer (one buffer
209 /// for equalities and one for inequalities). The size of each buffer is
210 /// numReservedCols * number of inequalities (or equalities). The reserved size
211 /// is numReservedCols * numReservedInequalities (or numReservedEqualities). A
212 /// coefficient (r, c) lives at the location numReservedCols * r + c in the
213 /// buffer. The extra space between getNumCols() and numReservedCols exists to
214 /// prevent frequent movement of data when adding columns, especially at the
215 /// end.
216 ///
217 /// The identifiers x_0, x_1, ... appear in the order: dimensional identifiers,
218 /// symbolic identifiers, and local identifiers. The local identifiers
219 /// correspond to local/internal variables created when converting from
220 /// AffineExpr's containing mod's and div's; they are thus needed to increase
221 /// representational power. Each local identifier is always (by construction) a
222 /// floordiv of a pure add/mul affine function of dimensional, symbolic, and
223 /// other local identifiers, in a non-mutually recursive way. Hence, every local
224 /// identifier can ultimately always be recovered as an affine function of
225 /// dimensional and symbolic identifiers (involving floordiv's); note however
226 /// that some floordiv combinations are converted to mod's by AffineExpr
227 /// construction.
228 ///
229 class FlatAffineConstraints {
230 public:
231  enum IdKind { Dimension, Symbol, Local };
232 
233  /// Constructs a constraint system reserving memory for the specified number
234  /// of constraints and identifiers..
235  FlatAffineConstraints(unsigned numReservedInequalities,
236  unsigned numReservedEqualities,
237  unsigned numReservedCols, unsigned numDims = 0,
238  unsigned numSymbols = 0, unsigned numLocals = 0,
239  ArrayRef<Optional<Value>> idArgs = {})
240  : numReservedCols(numReservedCols), numDims(numDims),
241  numSymbols(numSymbols) {
242  assert(numReservedCols >= numDims + numSymbols + 1);
243  assert(idArgs.empty() || idArgs.size() == numDims + numSymbols + numLocals);
244  equalities.reserve(numReservedCols * numReservedEqualities);
245  inequalities.reserve(numReservedCols * numReservedInequalities);
246  numIds = numDims + numSymbols + numLocals;
247  ids.reserve(numReservedCols);
248  if (idArgs.empty())
249  ids.resize(numIds, None);
250  else
251  ids.append(idArgs.begin(), idArgs.end());
252  }
