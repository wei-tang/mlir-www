Detailed Description

Simple C lists for non-owning mlir Opaque C types.

Recommended usage is construction from the data() and size() of a scoped owning SmallVectorImpl<...> and passing to one of the C functions declared later in this file. Once the function returns and the proper EDSC has been constructed, resources are freed by exiting the scope.
