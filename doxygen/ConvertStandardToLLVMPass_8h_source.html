79 /// Creates a pass to convert operations to the LLVMIR dialect. The conversion
80 /// is defined by a list of patterns obtained during the pass using the provided
81 /// callback and an optional type conversion class, an instance is created
82 /// during the pass.
83 /// By default stdlib malloc/free are used for allocating MemRef payloads.
84 /// Specifying `useAlloca-true` emits stack allocations instead. In the future

87 std::unique_ptr<OpPassBase<ModuleOp>>

89  bool useAlloca = false) {

92  [](MLIRContext *context) {

95  useAlloca);
