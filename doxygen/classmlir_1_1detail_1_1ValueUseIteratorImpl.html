Detailed Description

template<typename DerivedT, typename OperandType>
class mlir::detail::ValueUseIteratorImpl< DerivedT, OperandType >

A base iterator class that allows for iterating over the uses of a value.

This is templated to allow for derived iterators to override specific iterator methods.
