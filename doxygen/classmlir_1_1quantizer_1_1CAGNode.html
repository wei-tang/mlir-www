Detailed Description

A node in the Constraint Analysis Graph.

Nodes are either anchors (representing results and operands) or constraints. Anchor nodes are connected to other anchor nodes via constraints. Nodes exist within graph slices, which are typically analyses attached to the function or module. Slices can contain other slices, which mirrors the nesting of analyses.

Nodes have directed relationships which propagate successor-ward when dirty. Relationships can be bi-directional, in which case, the constraint's propagation mechanism must ensure convergence.
