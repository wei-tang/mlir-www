Detailed Description

The "affine.store" op writes an element to a memref, where the index for each memref dimension is an affine expression of loop induction variables and symbols.

The 'affine.store' op stores a new value which is the same type as the elements of the memref. An affine expression of loop IVs and symbols must be specified for each dimension of the memref. The keyword 'symbol' can be used to indicate SSA identifiers which are symbolic.
