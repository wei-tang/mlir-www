Detailed Description

template<typename ConcretePass>
struct mlir::PassRegistration< ConcretePass >

PassRegistration provides a global initializer that registers a Pass allocation routine for a concrete pass instance.

The third argument is optional and provides a callback to construct a pass that does not have a default constructor.

Usage:

/// At namespace scope. static PassRegistration<MyPass> reg("my-pass", "My Pass Description.");
