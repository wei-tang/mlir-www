Detailed Description

This class implements a command-line parser for MLIR passes.

It registers a cl option with a given argument and description. This parser will register options for each of the passes and pipelines that have been registered with the pass registry; Meaning that -cse will refer to the CSE pass in MLIR. It also registers an argument, pass-pipeline, that supports parsing a textual description of a pipeline.
