Detailed Description

template<typename OperandType>
class mlir::FilteredValueUseIterator< OperandType >

This class represents an iterator of the uses of a IR object that optionally filters on a specific sub-value.

This allows for filtering the uses of an IRMultiObjectWithUseList.
